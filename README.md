# SlamRobot
2021 年 robo-master 自动步兵机器人

## 预期目标
1. 实时生成 2d 栅格地图，保存地图，加载现有的地图。
2. 自主规划运动路径，在遇到动态障碍物时重新决策。
3. 根据外部指令调整运动策略。

## 可用传感器
- 2D 雷达(思岚 `Delta2a`)
- 双目相机(待定)
- UWB

## 主要结构
1. 建图和定位：`Cartographer`, `OpenVSLAM`(备选)
2. 里程计：待定
3. 路径规划：`ROS Navigation`(筹)
4. 仿真环境：`V-REP`(筹)
